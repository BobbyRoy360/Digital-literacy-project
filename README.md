Digital Literacy Project
Author: Abhijeet Wadnere | 25BAI11429 |
Overview
This project explores digital literacy through five practical tasks — covering infographic design, professional platform building, Python fundamentals, professional communication, and cybersecurity awareness.
Tasks
Task 1 – Digital Literacy Infographic
Designed an infographic titled "Digital Literacy Demystified" using Canva, covering the three pillars of digital literacy: Critical Thinking, Technical Skills, and Creative Expression — along with cybersecurity tips like MFA, link verification, and the "Front Page Test."
Task 2 – Professional Platform Strategy
Built a professional presence on GitHub (BobbyRoy360), LinkedIn, and Kaggle, with a four-year roadmap targeting consistent contributions, Kaggle ranking, and internship readiness by 2029.
Task 3 – Python Fundamentals & Digital Awareness Quiz
Practiced Python on CodeChef (arithmetic, print() function) and completed a Digital Literacy Awareness Quiz on Google Forms testing phishing detection and cybersecurity knowledge.

Task 4 – Professional Communication
